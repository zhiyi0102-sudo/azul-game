# 花砖物语测试用例

基于 GAME_RULES.md 文档的完整测试用例。

### 运行方式
```bash
cd ~/dev/azul-game
npm test
```

---

## 测试用例

### 1. 游戏初始化测试

**1.1 工厂数量（根据玩家人数）**
- [ ] 2人游戏：5个工厂
- [ ] 3人游戏：7个工厂  
- [ ] 4人游戏：9个工厂

**1.2 花砖袋**
- [ ] 100块花砖（5种颜色各20块）

**1.3 玩家模板**
- [ ] 5行图案线（1,2,3,4,5格）
- [ ] 5x5墙面
- [ ] 7格地板区

---

### 2. 工厂选取阶段测试

**2.1 从工厂拿砖**
- [ ] 只能拿同一种颜色的所有花砖
- [ ] 拿走后工厂剩余花砖移到中央

**2.2 从中央拿砖**
- [ ] 只能拿同一种颜色的所有花砖

**2.3 起始玩家标记**
- [ ] 从中央拿砖时拿取起始标记
- [ ] 起始标记占1格地板位置

**2.4 放置规则**
- [ ] 只能放入有空位的图案线
- [] 同图案线只能放同颜色
- [] 超过容量放入地板

**2.5 重要限制**
- [ ] 不能放入墙面已有该颜色的行

---

### 3. 地板区测试

**3.1 扣分规则**
- [ ] 第1块砖：-1分
- [ ] 第2块砖：-2分
- [ ] 第3块砖：-3分
- [ ] 第4块砖：-4分
- [ ] 第5块砖：-5分
- [ ] 第6块砖：-6分
- [ ] 第7块砖：-7分

**3.2 起始标记位置**
- [ ] 起始标记占1格

---

### 4. 墙面铺设阶段测试

**4.1 移动顺序**
- [ ] 从上到下检查
- [ ] 移动最右侧花砖到墙面

**4.2 计分规则（即时）**
- [ ] 无相邻：1分
- [ ] 横向相邻：横向连续数
- [ ] 纵向相邻：纵向连续数
- [ ] 同时有横向和纵向：横向+纵向

**4.3 转移规则**
- [ ] 完成的图案线移到墙面
- [ ] 剩余空图案线保留
- [ ] 空的图案线移除剩余花砖

---

### 5. 游戏结束测试

**5.1 结束条件**
- [ ] 有玩家完成1条完整横行时结束

**5.2 最终奖励分**
- [ ] 完整横行：+2分
- [ ] 完整竖列：+7分
- [ ] 同色5块：+10分
